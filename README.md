# RBaritone Lite — Fabric 1.21.11

Это не полный Baritone, а лёгкий клиентский автопилот для Fabric 1.21.11.
Он умеет идти к координатам, следовать за игроком и останавливаться.

## Команды в игре

```text
/rbar
/rbar goto <x> <y> <z>
/rbar follow <nick>
/rbar stop
/rbar status
```

## Что умеет

- Поворачивает камеру в сторону цели.
- Нажимает движение вперёд.
- Прыгает, если упёрся в блок или цель выше.
- Бежит, если цель далеко.
- Следует за игроком по нику, если игрок виден клиенту.

## Чего пока нет

- Умного A* пути как в настоящем Baritone.
- Копания блоков.
- Автостроительства.
- Обхода сложных препятствий, лавы, пропастей и античита.

## Сборка через GitHub

1. Создай новый репозиторий на GitHub.
2. Загрузи туда все файлы из этой папки.
3. Открой вкладку `Actions`.
4. Запусти workflow `Build Fabric Mod`.
5. Готовый `.jar` будет в `Artifacts`.

## Локальная сборка

Нужны Java 21 и Gradle 9.2+.

```bash
gradle build
```

Готовый мод появится в:

```text
build/libs/rbaritone-lite-1.0.0.jar
```

## Установка

1. Установи Fabric Loader для Minecraft 1.21.11.
2. Положи `fabric-api` под 1.21.11 в папку `mods`.
3. Положи `rbaritone-lite-1.0.0.jar` в папку `mods`.
4. Запусти игру.

Используй только в одиночной игре или на своих серверах, где такая автоматизация разрешена.
